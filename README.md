# Perú Exporta — CRM Panel

Panel de ventas para gestión de leads de WhatsApp.

## Stack
- React + Vite
- CSS Modules
- Apps Script como API backend

## Deploy en Vercel

1. Sube este repo a GitHub
2. Entra a vercel.com → New Project → importa el repo
3. Framework: Vite (lo detecta automático)
4. Deploy

## Desarrollo local

```bash
npm install
npm run dev
```

## Variables
La URL de Apps Script está en `src/api.js`
